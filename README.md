# @durkdotdev/configs

Configuration files for developing with commitlint, ESLint, lint-staged, Prettier, and TypeScript.

## Installation

```bash
npm install @durkdotdev/configs -D
# or
yarn add @durkdotdev/configs -D
# or
pnpm add @durkdotdev/configs -D
```
